# LomaSihteeri Master Overview

Tämä hakemisto sisältää kaiken LomaSihteeri-ominaisuutta koskevan suunnittelu-, konsepti- ja tuotedokumentaation. Nämä dokumentit toimivat projektin "Single Source of Truth" (SSOT) -lähteenä.

## Perussääntö

**Kaiken kehitystyön on noudatettava näitä dokumentteja. MVP-rajauksesta ei saa poiketa ilman, että dokumentaatiota päivitetään ja hyväksytetään ensin.**

## Dokumentit

- **[concept.md](./concept.md):** Kuvaa ongelman, ratkaisun, ydinlupauksen ja palvelun asemoinnin.
- **[mvp.md](./mvp.md):** Määrittelee MVP-version tarkan sisällön, toiminnallisuudet ja onnistumisen mittarit.
- **[daily_briefing.md](./daily_briefing.md):** Sisältää päivittäisen lomakirjeen rakenteen, sisältömallit ja kuratointiperiaatteet.
- **[ui_placement.md](./ui_placement.md):** Kuvaa, miten ja missä LomaSihteeri näkyy Lomaopas.fi-sivustolla.
- **[market_research.md](./market_research.md):** Tiivistelmä markkinatutkimuksesta, kilpailijoista ja ongelman validoinnista.
- **[monetization.md](./monetization.md):** Suunnitelma palvelun kaupallistamisesta MVP-vaiheen jälkeen.
- **[risks.md](./risks.md):** Tunnistetut riskit ja niiden hallintakeinot.
- **[brand.md](./brand.md):** Määrittelee palvelun nimen, brändisuhteen ja äänensävyn.
- **[pitch.md](./pitch.md):** Valmiita tiivistelmiä ja "hissipuheita" palvelusta.
